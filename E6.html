Présentation synthétique de deux projets évalués.

Projet 1 — Contrôleur de domaine et intégration (travail individuel)

Installation d’un serveur Windows Server (contrôleur de domaine) avec services essentiels (DNS, DHCP, AD DS). Promotion du serveur en contrôleur de domaine, création d’unités d’organisation et de comptes utilisateurs.

Intégration d’un second serveur dans le domaine du contrôleur principal, configuration des services partagés, puis création et ajout d’un poste client au domaine pour valider l’authentification centralisée et les GPO simples.

  • Rôle : administration système, configuration AD et services réseau.
  • Technologies : Windows Server 2022, Active Directory, DNS, DHCP, virtualisation (proxmox).

Projet 2 — Mise en place de GLPI (projet de groupe)

Installation et configuration d’un serveur GLPI pour la gestion de parc matériel et des tickets. Travail en groupe : Configuration LDAP et activation des inventaires - Déploiement des agents GLPI par GPO.

Configuration de l'authentification LDAP/Active Directory, Synchronisation des utilisateurs et groupes AD, Déploiement des agents GLPI sur les serveurs et les postes de travail via GPO, Vérification de la bonne remontée des inventaires dans la console GLPI.

  • Rôle : déploiement serveur, intégration GLPI, tests d’inventaire et documentation en équipe.
  • Technologies : Linux (Debian), GLPI, Active Directory.
